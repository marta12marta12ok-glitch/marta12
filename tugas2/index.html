01. Pengertian OOP

Object Oriented Programming (OOP) Object Oriented Programming (OOP) adalah paradigma pemrograman yang berorientasi pada objek, dimana program dibuat menggunakan class dan objek. Dalam OOP terdapat beberapa konsep utama seperti Encapsulation, Inheritance, Polymorphism, dan Abstraction yang membantu membuat program lebih terstruktur, modular, dan mudah dikembangkan.

02. Konsep Dasar

  • Class: Blueprint atau cetakan untuk membuat objek.
  • Object: Instance nyata yang tercipta dari sebuah class.
  • Attribute: Representasi data melalui variabel di dalam class.
  • Method: Logika fungsional atau fungsi di dalam class.

03. Class Sederhana (Car)

Pada contoh program di bawah ini terdapat class Car yang memiliki dua atribut yaitu model dan color yang digunakan untuk menyimpan data mobil. Selain itu terdapat dua method yaitu start() dan stop() yang berfungsi untuk menjalankan dan menghentikan mobil.

DART SOURCE CODE
// Nama   : Marta saputra
// Nim    : 251410020
// Kelas  : SI2A

class Car {
  String? model;
  String? color;
  void start() {
    print('Car started!');
  }
  void stop() {
    print('Car stopped!');
  }
}
void main() {
  Car myCar = Car();
  
  myCar.model = "Sedan";
  myCar.color = "Merah";
  myCar.start();
  myCar.stop();
}

04. Object Instantiation

Object Instantiation adalah proses membuat objek dari sebuah class. Pada contoh berikut dibuat sebuah objek bernama myCar dari class Car menggunakan perintah Car myCar = Car();. Setelah objek dibuat, atribut model dan color diisi dengan nilai "Sedan" dan "Merah".

DART SOURCE CODE
// Nama   : Marta saputra
// Nim    : 251410020
// Kelas  : SI2A

class Car {
  String? model; 
  String? color; 
}

void main() {
  Car myCar = Car();
  
  myCar.model = "Sedan"; 
  myCar.color = "Merah";

  print(myCar.model); 
  print(myCar.color);
}

05. Constructor

Constructor adalah metode khusus dalam sebuah class yang digunakan untuk menginisialisasi nilai atribut saat objek dibuat. Pada contoh ini, constructor Car(this.model, this.color) digunakan untuk langsung memberikan nilai pada atribut model dan color ketika objek myCar dibuat.

DART SOURCE CODE
// Nama   : Marta saputra
// Nim    : 251410020
// Kelas  : SI2A

class Car {
  String model; 
  String color;

  Car(this.model, this.color);

  void start() {
    print('Engine Online: $model'); 
  }
}

void main() {
  Car myCar = Car("Sedan", "Merah");

  myCar.start(); 
  print(myCar.color); 
}

07. Encapsulation

Encapsulation adalah konsep OOP untuk menyembunyikan data di dalam class agar tidak bisa diakses langsung dari luar. Akses ke data tersebut dilakukan melalui method seperti getter dan setter untuk menjaga keamanan dan kontrol terhadap data.

DART SOURCE CODE
// Nama   : Marta saputra
// Nim    : 251410020
// Kelas  : SI2A

class Person {
  String _name; 
  int age;    

  Person(this._name, this.age);

  void setName(String name) => _name = name;
  
  String getName() => _name;
}

void main() {
  Person person = Person("Saputra Marta", 18);

  print(person.age); 
  person.setName("Marta saputra");
  print(person.getName()); 
}

08. This Keyword

this adalah keyword yang digunakan untuk merujuk pada instance (objek) dari class saat ini. Biasanya digunakan di dalam constructor atau method untuk mengakses field atau method milik class tersebut.

DART SOURCE CODE
// Nama   : Marta saputra
// Nim    : 251410020
// Kelas  : SI2A

class Product {
  String nama; 
  Product(this.nama);
}

void main() {
  Product produk1 = Product("Laptop Gaming");
  print(produk1.nama); 
}

09. Product Class

Class Produk menggunakan named parameters pada constructor sehingga setiap nilai dapat diberikan dengan nama parameternya. Beberapa parameter menggunakan required agar wajib diisi saat membuat objek, sedangkan parameter lain memiliki nilai default.

DART SOURCE CODE
// Nama   : Marta saputra
// Nim    : 251410020
// Kelas  : SI2A

class Produk {
  final String id, nama; 
  final double harga; 
  final int stok;   
  final String kategori;

  Produk({
    required this.id,
    required this.nama,
    required this.harga,
    this.stok = 0,
    this.kategori = 'Umum',
  });
}

void main() {
  Produk laptop = Produk(
    id: "P001",
    nama: "Laptop Gaming",
    harga: 15000000,
    stok: 15,
    kategori: "Elektronik",
  );

  Produk smartphone = Produk(
    id: "P002",
    nama: "Smartphone Flagship",
    harga: 12000000,
    stok: 25,
  );

  Produk kamera = Produk(
    id: "P003",
    nama: "Kamera Mirrorless",
    harga: 20000000,
  );

  print(laptop.nama); 
  print(smartphone.nama); 
  print(kamera.nama); 
}
