1. Class & Object

Class adalah blueprint untuk membuat object, sedangkan object adalah hasil dari class.

2. Constructor

Constructor digunakan untuk menginisialisasi object saat dibuat.

3. Named Constructor

Constructor dengan nama tambahan agar bisa membuat object dengan cara berbeda.

4. Constant Constructor

Digunakan untuk membuat object yang tidak berubah (immutable).

5. Initializer List

Digunakan untuk menginisialisasi nilai sebelum constructor dijalankan.

6. Enkapsulasi

Enkapsulasi adalah konsep untuk melindungi data dengan menggunakan akses terbatas (private) agar tidak bisa diakses langsung dari luar class.

7. Getter

Getter digunakan untuk mengambil nilai dari variabel private secara aman.

8. Setter

Setter digunakan untuk mengubah nilai variabel private dengan aturan tertentu.

9. Validasi Data

Validasi digunakan untuk memastikan data yang dimasukkan sesuai aturan agar tidak terjadi kesalahan.

10. Sistem Login

Sistem login digunakan untuk autentikasi pengguna dengan mencocokkan username dan password.

11. Rekening Bank

Konsep class digunakan untuk membuat sistem sederhana seperti rekening bank dengan fitur saldo, deposit, dan penarikan.

12. Studi Kasus Produk

Contoh penggunaan getter dan setter dalam pengelolaan data produk dengan validasi.

13. Studi Kasus User

Penerapan OOP dalam sistem user dengan validasi data seperti username, password, dan email.

14. Sistem Login & Ganti Password

Implementasi autentikasi login serta fitur perubahan password dengan validasi keamanan.
