1. Class & Object

Class adalah blueprint untuk membuat object, sedangkan object adalah hasil dari class.

2. Constructor

Constructor digunakan untuk menginisialisasi object saat dibuat.

3. Named Constructor

Constructor dengan nama tambahan agar bisa membuat object dengan cara berbeda.

4. Constant Constructor

Digunakan untuk membuat object yang tidak berubah (immutable).

5. Initializer List

Digunakan untuk menginisialisasi nilai sebelum constructor dijalankan.
